# KNN Handwritten Digit Classification

This project uses K-Nearest Neighbors (KNN) to classify handwritten digits from sklearn's digits dataset.

Steps:
- Loaded digits dataset
- Visualized sample images
- Applied train-test split
- Scaled features using StandardScaler
- Trained KNN with K values 3,5,7,9
- Selected best K using accuracy
- Plotted Accuracy vs K
- Generated confusion matrix
- Displayed predictions

Result:
Model successfully classifies digits using distance-based learning.
